フランジの種類と特徴
差込み溶接式フランジ（Slip-on Flange）
  • ハブなし板フランジ（SOP：Slip On Plate）
    平坦な板状でハブを持たない簡素な構造。
    製造コストが低く、取り扱いも容易なため、主に建築配管・上下水道・一般設備配管などの低中圧ラインで使用されます。
    ASME規格ではClass 150程度までの圧力で使用されることが一般的です。
  • ハブ付きフランジ（SOH：Slip On Hub）
    フランジ中心にハブ（突出したボス）があり、パイプを差し込んで突合せおよび角溶接により固定します。
    軸方向の応力に強く、比較的高圧の配管や振動の多いライン（例：蒸気配管、空調配管）で使用されます。
    SOPよりも強度が高く、標準的にはClass 300前後まで対応可能です。
接合面（シート面）の形状
  • FF（Flat Face：フラットフェース）
    接合面が完全に平らな構造で、主に鋳鉄製フランジや低圧用途（例：冷却水ライン、排水系）で使用されます。
    非金属系ソフトガスケット（例：ノンアスベスト、ゴム）との相性が良く、面圧の分散によってフランジやパイプへの局部的応力集中を防ぎます。
    通常はASME Class 125〜250程度に対応。
  • RF（Raised Face：レイズドフェース）
    接合面の中央部が高くなっており、ガスケットが突起部で集中圧縮されるため密封性が高い構造です。
    石油化学・発電・高温高圧の配管システムなど、広範な産業で最も一般的に採用されている形状です。
    使用圧力はClass 150〜2500まで幅広く、金属系スパイラルワウンドガスケットなどと併用されます。
  • RTJ（Ring Type Joint）
    RFタイプの中心部に金属ガスケットを装填する溝が設けられた構造で、非常に高い密封性能を発揮します。
    高温・高圧条件（例：900クラス以上、750℃超）下での使用に最適で、原油・ガス・高圧蒸気などのラインで採用されます。
    ガスケットはOCTG（石油・ガス掘削設備）でも使用されるRタイプ・RXタイプなどが主流です。
閉止フランジ（Blind Flange / BF）
  • BFフランジ（Blind Flange または Blank Flange）
    中央に穴がなく、流体を完全に遮断できる厚板状のフランジです。
    配管の終端を塞ぐために用いられ、将来の延長工事やメンテナンスのために一時的に閉止する場面でも利用されます。
    耐圧性能が高く、水圧試験（耐圧試験）や空気漏洩試験時の一時閉止としても重宝されます。
    圧力クラスはFF/RF面どちらにも対応し、ASMEでは150〜2500までの等級が規格化されています。
ラップジョイントフランジ（Lap Joint Flange / LJF）
  • LJF（Lap Joint Flange）
    単体ではシール性を持たず、スタブエンドと組み合わせて使用する可動式フランジです。
    本体のフランジは締結のみに用いられ、シール面はスタブエンドの先端部で担います。
    分解・再組立てが多いライン（例：タンク接続部、サニタリー配管）や腐食性流体で非金属パイプと金属フランジを組み合わせる場合にも活用されます。
    圧力用途としてはClass 150〜300程度が一般的です。
突合せ溶接式フランジ（Weld Neck Flange / WNF）
  • WNF（Weld Neck Flange）
    フランジに長いテーパーハブがあり、配管と突合せ溶接で接続する構造です。
    接続部の応力分布が滑らかで、高圧・高温・振動の激しい環境に非常に適しており、原子力・石油化学・火力発電などの重要配管に使用されます。
    対応圧力はASME Class 150〜2500、高温用鋼材（例：SUS304, SUS316, SCM）などとも組み合わせ可能です。
ねじ込み式フランジ（Threaded Flange / THF）
  • THF（Threaded Flange）
    フランジ内面にねじ加工（NPTなど）が施されており、パイプをねじ込んで接続します。
    溶接を行わずに接続できるため、可燃性ガス配管や一時設置の現場などで便利です。
    ただし、構造上の強度は溶接式より劣るため、低圧・小口径配管（主に25A〜100A程度）に限定されます。
ソケット溶接式フランジ（Socket Weld Flange / SWF）
  • SWF（Socket Weld Flange）
    フランジ中心にソケット形状の穴があり、パイプを差し込んで角溶接で接続する構造。
    小口径での使用が多く、突合せ溶接に比べて施工が簡単かつ短時間で完了します。
    主にClass 300以下の高圧・高温用途（例：冷却水ライン、蒸気ライン）で使用されます。
    注意点として、流体抵抗が増す可能性があるため、連続的な流れが要求される箇所では不向きです。
